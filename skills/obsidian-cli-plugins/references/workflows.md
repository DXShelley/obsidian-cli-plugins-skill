# Obsidian Workflow Index

This compatibility index replaces the old combined runtime/task guide. Load the smallest specific file instead of this index when possible:

- `fast-paths.md`: common request routing and high-frequency commands.
- `runtime-sync.md`: runtime configuration, vault resolution, Git sync, cross-agent contract, plugin command IDs, manual command execution, and official CLI lookup.
- `vault-safety.md`: safe read/search/summarization of arbitrary vault content.
- `task-query.md`: task query/report workflows.
- `task-add.md`: task creation workflows.
- `record-workflows.md`: record creation workflows.
- `openclaw.md`: OpenClaw install/sync and staged/cross-agent media workflows.
- `official-cli.md`: native Obsidian CLI command syntax and risk notes.
- `extensions.md`: adding workflow support for newly installed plugins.

Keep old references to `workflows.md` working by using this routing map, but do not add new detailed workflow text here.
