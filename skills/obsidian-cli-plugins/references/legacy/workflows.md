# Obsidian Workflow Index

This compatibility index replaces the old combined runtime/task guide. Load the smallest specific file instead of this index when possible:

- `references/core/fast-paths.md`: common request routing and high-frequency commands.
- `references/core/runtime-sync.md`: runtime configuration, vault resolution, Git sync, cross-agent contract, plugin command IDs, manual command execution, and official CLI lookup.
- `references/core/vault-safety.md`: safe read/search/summarization of arbitrary vault content.
- `references/features/task-query.md`: task query/report workflows.
- `references/features/task-add.md`: task creation workflows.
- `references/features/record-workflows.md`: record creation workflows.
- `references/integrations/openclaw.md`: OpenClaw install/sync and staged/cross-agent media workflows.
- `references/integrations/official-cli.md`: native Obsidian CLI command syntax and risk notes.
- `references/maintenance/extensions.md`: adding workflow support for newly installed plugins.

Keep old workflow index use cases working by using this routing map, but do not add new detailed workflow text here.
